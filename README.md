# UrBetterHouse Project

A house marketplace PoC written in Ruby on Rails.

## Requirements

* Ruby 2.7.2
* Rails 6.0.5
* Node 12.22.12
* Yarn 1.22.19

## Installation

```bash
# rails db:migrate
# rails scraper:ur_house
# rails ur_better_house:demo_accounts
```
